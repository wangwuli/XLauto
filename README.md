# XLauto

动机/Motivation<br>
结合个人的一些经历，开发的一套运维系统，便于日常的工作，也对一些脚本以及程序进行总结与记录。

依赖/dependency<br>
node.js、python<br>

初始化/initialization<br>
python install -r main/requirements.txt<br>
cd ./frontend<br>
npm -i <br>

启动命令/start command<br>
cd ./frontend & npm run serve<br>
cd ./main & python manage.py<br>
